💎 Jewellery E-Commerce Website

A modern and responsive jewellery website built using React, designed to showcase elegant collections and provide a smooth shopping experience for users.

✨ Overview

This project is a fully functional front-end jewellery e-commerce platform where users can explore different jewellery collections such as rings, necklaces, earrings, and more. The website focuses on a premium UI/UX design with smooth navigation and attractive layouts.

🚀 Features
🛍️ Beautiful product listings and collections
🔍 Product details view
🧭 Smooth navigation with React Router
🎨 Modern and responsive UI design
⚡ Fast performance using React & Vite
🔝 Scroll-to-top functionality
💫 Animations using Lottie (optional)
🛠️ Tech Stack
Frontend: React.js
Styling: CSS / Tailwind CSS
Routing: React Router
Icons: Lucide React / React Icons

🛍️ Core Functionality

The website provides essential e-commerce features such as:

Product Listings: Displays various jewellery collections in an organized grid format.
Product Details Page: Each item includes detailed information such as name, price, description, and images.
Category Browsing: Users can explore jewellery based on types or collections.
Smooth Navigation: Implemented using modern routing techniques for a seamless browsing experience.
Interactive UI Elements: Hover effects, animations, and dynamic components enhance user interaction.


# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration
